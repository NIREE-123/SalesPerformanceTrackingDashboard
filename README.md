# SalesPerformanceTrackingDashboard
This Excel project analyzes 125+ office product sales (laptops, desks, printers, chairs, mobiles) tracking ID, date, customer, region (North/South/East/West), salesperson, qty (1-7), price (₹7.5k-55k), discount (0-8%), &amp; status. Revenue for Paid only: Net = Qty × Price × (1-Discount/100).
